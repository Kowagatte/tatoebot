### Installing a bot using Tatoebot Source
`Install Node js`

Install `npm install discord.js`
Install `npm install xmlhttprequest`
Install [Google Cloud SDK](https://cloud.google.com/nodejs/docs/setup)

Goto Source and run `node bot.js`
